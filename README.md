### Tech Stack

- AWS (Amazon Web Services)
  - ApiGateway (Websocket API)
  - SageMaker (DeepAR Forecasting Algorithm)
  - Lambda
  - Comprehend (Sentiment Analysis)
  - S3 Buckets (Datasets & Hosting)
  - DynamoDb (Backend Database)
- React
- Typescript
- Open Data API
- TailwindCss
- Twitter API V2
